gCalBirthdays - README
July 03, 2009

Overview
========
gCalBirthdays is a Java Application which uses 
the Google data APIs ("GData" for short).
It reads Your Google Contacts extract the birthdays and add them to Google Calendar.  


Package information
===================
This package contains:
 o  Java source code 
 o  Runnable JAR file
 o  README
 o  INSTALL
 o  RELEASE_NOTES

Refer to INSTALL.txt for details on how to install and use this package.
